Switch it Up!

When provided with a number between 0-9, return it in words.


My code:

function switchItUp(number) {
                        switch(number) {
                          case 0:
                            return "Zero";
                            break;
                          case 1:
                            return "One";
                            break;
                          case 2:
                            return "Two"
                            break;
                          case 3:
                            return "Three";
                            break;
                          case 4:
                            return "Four";
                            break;
                          case 5:
                            return "Five";
                            break;
                          case 6:
                            return "Six";
                            break;
                          case 7:
                            return "Seven";
                            break;
                          case 8:

And that's the best solution among all users:

function switchItUp(number){
                        switch(number) {
                          case 0:
                            return 'Zero';
                          case 1:
                            return 'One';
                          case 2:
                            return 'Two';
                          case 3:
                            return 'Three';
                          case 4:
                            return  'Four';
                          case 5:
                            return 'Five';
                          case 6:
                            return 'Six';
                          case 7:
                            return 'Seven';
                          case 8:
                            return 'Eight';
                          case 9:
                            return 'Nine';
                          default:
                            return 'Unknown number';
                        }
                      }
