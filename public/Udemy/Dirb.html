Dirb

The 'dirb' command is a web content scanner designed for directory enumeration. It is widely used to uncover hidden files and directories on a web server through brute-forcing directory names. I can use ‘dirb’, for example, on my blog by typing the command ‘dirb http://142.250.185.129’, and 'dirb' will start to enumerate all kinds of possible directories.

'Dirb' enumerated 4612 words to find a hidden directory on my blog. Currently, none have been found because the code is not 200, indicating that no connection could be established.

You can customize the dirb enumeration by entering 'dirb http://IPAddress /usr/share/wordlists/', then pressing the 'Tab' key twice, selecting 'dirb@', and hitting 'Enter'.

Now, I can choose one of those text files to search for hidden directories. If I don't manually select a text file, the default is common.txt.
