Active Information Gathering

Tools that can be used to gather information about a website include:

  • nslookup:
    This command is a network administration tool used in the command line to query the Domain Name System (DNS). It retrieves the mapping between a domain name and its IP address, along with other DNS records.
  • whois:
    The 'whois' command in Linux is used to obtain information about a domain, including the domain's owner, the contact details of the owner, and the nameservers assigned to the domain.
  • whatweb:
    WhatWeb identifies various web technologies such as content management systems (CMS), blogging platforms, statistical/analytics packages, JavaScript libraries, web servers, and embedded devices. Documenting gathered information is always beneficial. To achieve this with WhatWeb, simply include "--log-verbose=nameOfFile" in the command line. For enhanced readability, you may include the "-v" option and the "--no-errors" flag.
  • theHaverster:
    TheHarvester is utilized for collecting open-source intelligence (OSINT) on a company or domain. From the command line, you can specify which search engine you prefer to use, such as Google, Bing, Yahoo, etc. You can discover many additional options by using 'theHarvester --help' command. For instance, the command 'theHarvester -d google.com -b all' could be used, where '-d' represents the domain and '-b' specifies the sources.
